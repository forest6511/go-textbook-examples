# 第5章 データ構造

「Go言語の教科書」第5章のサンプルコードです。

## 実行方法

```bash
go run main.go
```

## 内容

- 配列とスライス（作成、`append`、スライス式、`copy`、内部構造）
- マップ（基本操作、comma-ok イディオム、ソート済み反復）
- 構造体（定義、初期化、埋め込み）
- Stack 実装（構造体 + メソッドの組み合わせ）
- ポインタ（基本、`new`、関数引数）
